# 体系框架

借款分为有息与无息的，有息的包括：贷款、发行债券；




# 补充框架

## 借款费用的确定
### 确认原则
可归属于资本化条件的资产构建或生产的，应资本化，计入相关资产成本；
其他费用，应当在发生的时候根据发生额确认为费用，计入当期损益；
### 资本化期间的三个问题
只有发生在资本化期间的有关费用才允许资本化
起点、暂停、终点 \
起点三个条件：支出、借款费用、开始建造 \
暂停两个条件：非正常中断、连续三个月
终止条件：符合出售的条件



## 会计处理
### 利息的资本化
利息净支出 = 专门借款：以专门借款当期实际利息-所剩借款的利息或者投资收益
