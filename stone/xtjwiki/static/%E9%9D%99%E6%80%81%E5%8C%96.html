How to build a TiddlyWiki5 from individual tiddlers

TiddlyWiki on Node.js

# 生成单个文件
tiddlywiki xtjwiki/ --rendertiddlers '[!is[system]]' $:/core/templates/static.tiddler.html static text/plain
# 直接生成当前的编辑视图
tiddlywiki xtjwiki/ --rendertiddler $:/core/templates/static.template.html static.html text/plain
# 生成css
tiddlywiki xtjwiki/ --rendertiddler $:/core/templates/static.template.css static/static.css text/plain
